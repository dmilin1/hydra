# Settings Overview

Hydra's settings are organized into categories, making it easy to find and configure the features you want. Access settings by tapping the **Settings** tab in the bottom tab bar. You can use the **search bar** at the top to ask a question or search for a specific setting.

## Settings Categories

### General

General settings cover core functionality including swipe gestures, post and comment sorting preferences, content filters, link handling, app startup behavior, external link browser selection, and legal information.

[View General Settings](hydra://settings/general) | [Learn More](hydra://settings/guide/?doc=general_settings)

### Guide

The Guide is Hydra's built-in documentation. Browse help articles by category or search for answers to your questions.

[View Guide](hydra://settings/guide) | [Learn More](hydra://settings/guide/?doc=getting_started)

### Theme

Choose a color scheme for Hydra. You can set separate themes for light and dark mode, create custom themes (Pro), or browse community-created themes.

[View Theme Settings](hydra://settings/theme) | [Learn More](hydra://settings/guide/?doc=themes)

### Appearance

Control how posts, comments, and the tab bar are displayed. Options include compact post mode, text line limits, flair visibility, NSFW/spoiler blurring, auto-play videos, Live Text, and more.

[View Appearance Settings](hydra://settings/appearance) | [Learn More](hydra://settings/guide/?doc=appearance_settings)

### App Icon

Change Hydra's home screen icon. Choose from alternate icons created by community members. This option only appears on iOS devices that support alternate icons.

[View App Icon Settings](hydra://settings/appIcon) | [Learn More](hydra://settings/guide/?doc=app_icons)

### Account

Manage your Reddit accounts. Switch between accounts, add new ones, or remove existing ones.

[View Account Settings](hydra://accounts) | [Learn More](hydra://settings/guide/?doc=accounts_and_login)

### Data Use

Enable low data mode separately for Wi-Fi and cellular connections. When enabled, videos only load on tap, article images are skipped, and subreddit icons are not loaded.

[View Data Use Settings](hydra://settings/dataUse) | [Learn More](hydra://settings/guide/?doc=data_use_settings)

### Stats

View your usage statistics including Reddit activity, favorite communities, and achievements. Stats is a Pro feature â€” non-subscribers can preview the page but data is obfuscated.

[View Stats](hydra://settings/stats) | [Learn More](hydra://settings/guide/?doc=stats)

### Privacy

Control whether Hydra sends error reports when it encounters a crash or loading issue. Error logs include a stack trace, your Reddit username (if logged in), and device details. This setting requires an app restart to take effect.

[View Privacy Settings](hydra://settings/privacy) | [Learn More](hydra://settings/guide/?doc=privacy_settings)

### Advanced

Advanced settings for power users. Clear the image cache, configure a self-hosted Hydra server, or view your customer ID.

[View Advanced Settings](hydra://settings/advanced) | [Learn More](hydra://settings/guide/?doc=advanced_settings)

### Hydra Pro

Subscribe to Hydra Pro or manage your existing subscription. Pro unlocks features like inbox alerts, gallery mode, stats tracking, AI-powered post filtering, post and comment summaries, and additional themes.

[View Hydra Pro](hydra://settings/hydraPro) | [Learn More](hydra://settings/guide/?doc=hydra_pro)

### Patch Notes

View what's new in the latest Hydra update.

### Request A Feature

Submit feature requests to the Hydra community on Reddit.

---

All settings save automatically as you change them and persist across app sessions. Some Pro features require a Hydra Pro subscription, and some features are only available on specific devices.
